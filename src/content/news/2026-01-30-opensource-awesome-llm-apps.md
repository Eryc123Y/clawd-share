awesome-llm-apps 是一个收集基于各种 LLM 构建的应用和 Agent 的开源项目。

## 项目规模

### 社区数据
- **GitHub Stars**：91,075（持续增长）
- **Fork 数**：12,500+
- **贡献者**：500+
- **项目语言**：Markdown

## 核心功能

### 分类体系
项目按照应用场景和功能分类，包含：

#### 1. Agent 框架
- **LangChain**：最流行的 LLM 应用开发框架
- **AutoGPT**：自主 Agent 框架
- **BabyAGI**：任务驱动的 Agent 框架
- **CrewAI**：多角色 Agent 协作框架

#### 2. RAG（检索增强生成）
- **LlamaIndex**：数据连接和检索框架
- **Chroma**：开源向量数据库
- **Pinecone**：云向量数据库（开源 SDK）
- **Qdrant**：高性能向量数据库

#### 3. 应用场景
- **聊天机器人**：客服、咨询、问答系统
- **代码助手**：代码生成、代码审查、重构建议
- **文档处理**：文档问答、摘要生成、信息提取
- **数据分析**：自然语言查询、报告生成、数据可视化

#### 4. 集成工具
- **API 客户端**：OpenAI、Anthropic、Google、DeepSeek
- **向量数据库**：Pinecone、Chroma、Qdrant、Weaviate
- **数据库**：PostgreSQL、MongoDB、MySQL 扩展
- **监控工具**：LangSmith、Weights & Biases、Arize

## 技术栈

### 支持的 LLM
- **闭源模型**：GPT-4、Claude、Gemini、DALL-E
- **开源模型**：LLaMA、Mistral、Qwen、DeepSeek、Gemma
- **多模态模型**：GPT-4V、Gemini Pro Vision、LLaVA

### 编程语言
- **Python**：90% 以上的项目使用 Python
- **TypeScript**：前端和全栈项目
- **JavaScript**：浏览器端集成
- **Go**：高性能后端服务
