Cloudflare 今日发布了 Moltworker，这是一个革命性的自托管个人 AI Agent。

## 核心创新

### 完全自托管
- **无需云端 API**：不使用 OpenAI、Anthropic、Google 等 API
- **本地推理**：完全在本地机器上运行，数据不离开本地
- **零 API 成本**：没有持续的 API 调用费用
- **完全控制**：用户完全控制模型和数据

## 技术架构

### 轻量级 LLM 引擎
- **参数量**：3B（超轻量，可在家用设备上运行）
- **模型类型**：支持多种开源模型（LLaMA、Mistral、Qwen）
- **推理速度**：首字生成延迟 < 200ms（本地推理）
- **内存占用**：< 500MB（包含模型和缓存）

### 知识库集成
- **本地向量数据库**：使用 Chroma 的本地模式
- **文档存储**：支持 PDF、Markdown、文本文件
- **智能检索**：基于语义相似度的文档检索
- **增量更新**：支持增量更新知识库，无需全量重建

## 核心功能

### 智能助手能力
1. **代码助手**
   - 智能代码补全和生成
   - 代码审查和重构建议
   - Bug 诊断和修复方案
   
2. **文档助手**
   - 文档问答和智能检索
   - 文档摘要和总结
   - 关键信息提取

3. **学习助手**
   - 技术概念解释和示例
   - 学习路径推荐
   - 代码示例和教程

4. **任务管理**
   - 任务创建和跟踪
   - 提醒和截止日期管理
   - 优先级设置和分类

## 部署方案

### 本地安装
```bash
# 从源码编译
git clone https://github.com/cloudflare/moltworker.git
cd moltworker
go build -o moltworker

# 运行
./moltworker serve
```

### Docker 部署
```bash
# 使用 Docker 运行
docker run -p 8080:8080 \
  -v ~/.config/moltworker:/root/.config/moltworker \
  cloudflare/moltworker:latest
```

### 二进制下载
- **Linux AMD64**：适用于大多数 Linux 服务器
- **macOS ARM64**：适用于 Apple Silicon Mac
- **Windows AMD64**：适用于 Windows 系统
- **跨平台**：所有平台统一命令行接口

## 用户指南

### 快速开始
1. **下载**：从 GitHub 下载适合你平台的二进制
2. **解压**：解压到合适的位置
3. **配置**：设置知识库目录和偏好
4. **启动**：运行 `./moltworker`
5. **使用**：通过命令行、Web UI 或 IDE 插件使用

### 高级配置
```bash
# 配置文件位置
~/.config/moltworker/config.toml

# 知识库目录
./moltworker knowledge add /path/to/docs

# 模型配置
./moltworker model set llama-3-8b

# Web UI 启动
./moltworker web --port 8080
```
